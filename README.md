# DIU-Campus-Sim-A-2D-Interactive-OpenGL-Environment
This project was developed as part of the Computer Graphics Lab (CSE422) curriculum. It showcases the practical application of mathematical concepts like Sine-wave oscillations for flag waving and Arc-logic for sun/moon movement, bridging the gap between theoretical geometry and digital visualization.
📝 Project Description
This project is a detailed 2D computer graphics simulation of the Daffodil International University (DIU) campus built using C++ and OpenGL (GLUT library). It features a vibrant, interactive environment that brings campus life to a digital canvas. The simulation includes key architectural landmarks like the Academic Building (AB-01), a realistic Mosque, and a high-detail Football Field.

The project goes beyond static visuals by implementing dynamic animations and environmental effects. It showcases technical proficiency in geometric transformations, primitive rendering, and real-time user interaction through keyboard events.
